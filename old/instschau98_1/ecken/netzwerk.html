Netzwerk und Linux

Eine Stärke von Linux ist die Integration in ein Netzwerk. Linux unterstützt dazu eine Vielzahl von Protokollen und Schnittstellen, wie z.B.:

  • TCP/IP
    Das wohl am weitesten verbreitete Protokoll für Netzwerke. Es stellt den Standard im Internet dar und wird von Linux in vollem Umfang unterstützt. Auch für die neueste Version IPv6 gibt es bereits Unterstützung.
     
  • IPX
    Von Novell verwendetes und propagiertes Protokoll. Wird hauptsächlich für die Anbindung von DOS- und Windows-Clients verwendet.
     
  • Appletalk
    Protokoll, welches überwiegend bei Apple Macintosh-Rechnern Verwendung findet.
     
  • PPP/SLIP
    PPP und SLIP wird für Einwahl-Verbindungen mit ISDN- oder Analogmodem genutzt. Linux kann dabei sowohl als Server als auch als Client eingesetzt werden.
     
Linux unterstützt nicht nur diese Protokolle, sondern bietet gleichzeitig noch ein hohes Maß an Funktionalität, so daß ein Linux-PC ohne weiteres als Router, Firewall, Gateway etc. eingesetzt werden kann, vor allem auch in heterogenen Umgebungen mit anderen Betriebssystemen. Gerade in heterogenen Netzwerken liegt die Stärke von Linux, das mit Hilfe kostenloser Zusatzprogramme wie Samba auch alle in der Windows-Welt üblichen Netzwerkprotokolle beherrscht.
