Datenbanken

Linux bietet ein sehr reichhaltiges Angebot an Datenbanken, vor allem im SQL-Bereich gibt es ein sehr großes Angebot. Zwei Beispiele, die wir hier vorstellen möchten:
 
  • Adabas D von der Software AG
    Bei Adabas D handelt es sich um eine kommerzielle SQL-Datenbank, die auch auf anderen Plattformen anzutreffen ist. Adabas D ist in verschiedenen Ausbaustufen erhältlich.
     
  • MySQL
    Bei MySQL handelt es sich um eine etwas einfachere und für Privatanwender freie Datenbank. Durch die freie Verfügbarkeit inkl. Quellcode entstanden viele Erweiterungen und Tools. MySQL wird auch gerne als Backend für Webanwendungen verwendet.
     
Daß Linux kein Außenseiter mehr im Bereich Datenbanken ist, wird durch die angekündigte Portierung von Informix, einer großen kommerziellen Datenbank, untermauert.
