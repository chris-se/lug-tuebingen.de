Linux als Server

Neben der umfangreichen Funktioalität von Linux auf den unteren Ebenen eines Netzwerkes, beherrscht Linux auch eine umfassende Anzahl von Serverdiensten. Dazu gehören praktisch alle Internetdienste, wie z.B.:

  • HTTP (WWW)
    Ein sehr großer Teil der WWW-Server im Internet läuft unter Linux mit Apache. Durch die weite Verbreitung unterstützt sogar Microsoft Apache/Linux mit Frontpage-Erweiterungen.
     
  • SMTP/IMAP/POP (E-Mail)
    Mit POP und IMAP können E-Mails abgeholt werden, SMTP dient zum Versenden von Mails.
     
  • Vieles mehr wie FTP (Dateitransfer), News, IRC (Chat),
    DNS (Nameserver)...
     
Sehr viele Linuxrechner werden auch als Fileserver eingesetzt. Linux stellt Dateien für sehr viele verschiedene Betriebssysteme bereit. So gibt es NFS (vorwiegend für Unix/Linux-Clients), SMB für Windows/OS/2-Client oder auch Appletalk für MacOS. Linux kann auch einen Novell Netware Fileserver ersetzen.
 
Ein sehr großer Vorteil von Linux ist die Multiuser-Fähigkeit. Unter Linux können viele Menschen gleichzeitig arbeiten, ohne daß sie sich gegenseitig stören.
 
Ein weiterer Punkt ist, daß diese Dienste alle kostenlos verfügbar sind.
