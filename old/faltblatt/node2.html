Hilfe von außerhalb

Auch wenn die lokale Dokumentation noch so ausführlich ist - Situationen, in denen der Linux-Einsteiger den Rat erfahrenerer Anwender benötigt, gibt es immer wieder. Viele Distributionen bieten eine Telefon-Hotline für den Einsteiger-Support an. Für Linux-User ohne E-Mail-Adresse oder Internet-Anschluß ist das meist die einzige Anlaufstelle. Alle anderen können aus einem reichhaltigen Fundus an Informationsquellen schöpfen:

Mailinglisten
Eine Mailingliste funktioniert wie ein Postverteiler. Sie schicken eine Mail mit Ihrer Frage an eine zentrale Adresse. Von dort wird Sie an alle Personen weitergesandt, die die jeweilige Mailingliste abonniert haben. (Beispiel: Um die Mailingliste der Linux User Group Tübingen zu abonnieren, senden Sie eine Mail mit dem Text "subscribe <Ihre E-Mail-Adresse>" an den Listen-Verwalter unter lug-tuebingen-request@mail500.uni-tuebingen.de. Eine Mail an sämtliche eingetragenen Listenmitglieder können Sie über die zentrale Adresse lug-tuebingen@mail500.uni-tuebingen.de verschicken.)

Sinnvolle erste Anlaufstellen sind beispielsweise:

  • Diverse Mailinglisten der jeweiligen Distributionen - mehr dazu im Handbuch oder auf der Homepage des Distributors.
  • Mailinglisten einer Linux User Group in der Nähe. (Beispiel: Via lug-tuebingen@mail500.uni-tuebingen.de erreichen Sie die Liste der Linux User Group Tübingen.)
  • Die Linux-Newbie-Liste linux-newbie@vger.rutgers.edu. Diese Liste ist englischsprachig.
Newsgroups
Englischsprachige Newsgroups rund um Linux finden Sie unter comp.os.linux, die deutschsprachigen Pendants sind unter de.comp.os.unix.linux eingeordnet. Mitunter empfiehlt sich auch in Blick in allgemeine Unix-Newsgroups unter comp.os.unix, beziehungsweise de.comp.os.unix.
WWW
Es gibt kaum ein Linux-Thema, zu dem keine Homepage existiert. Hier nur eine kleine Auswahl mit unseren Lieblingsseiten:
  • www.linuxhq.com - Das Linux-,,Hauptquartier``.
  • metalab.unc.edu/LDP - Homepage des Linux Documentation Project, das zum Beispiel HOWTOs und Manpages koordiniert.
  • www.linux.org.uk - Allgemeine Linux-Informationen aus erster Hand von Kernel-Hacker Alan Cox.
  • www.kernel.org - Hauptseite für den Linux-Kernel.
  • www.linux.de - Deutschsprachige Linux-Informationen.
  • www.tuebingen.linux.de - Homepage des Linux User Group Tübingen.
  • www.slashdot.org - Aktuellste Nachrichten aus der und für die Linux-Szene.
  • www.freshmeat.net - Ankündigungen neuer Programme und Programmversionen.
  • www.linux-magazin.de - Homepage der einzigen deutschsprachigen Linux-Zeitschrift.
  • www.freibergnet.de - Vertrieb von Linux-Devotionalien. ;-)
IRC
Wenn es wirklich schnell gehen muß, sind die Chatkanäle #linuxger und #linux.de eine gute Adresse. Beachten Sie: Linux-Experten sind gerne im Streß und lieben kurze, prägnante Fragen. Wer erst fragt, ob er fragen darf, ist meist schon unten durch.
