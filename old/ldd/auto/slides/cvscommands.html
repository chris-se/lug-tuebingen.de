Kommandoübersicht

  • cvs --help-commands für eine Kommandoliste
  • cvs --help für globale Optionen
  • cvs --help kommando für Hilfe zu einem Kommando
  • cvs add files um files zum Projekt hinzuzufügen
  • cvs checkout projekt um eine Arbeitskopie von projekt zu erstellen
  • cvs commit um lokale Änderungen in das Repository zu übertragen
  • cvs remove files um files aus dem Projekt zu entfernen, files müssen zuerst gelöscht werden!
  • cvs tag name um der aktuellen Revision den symbolischen Namen name zu geben
  • cvs update um die Arbeitskopie auf den aktuellen Stand zu bringen
